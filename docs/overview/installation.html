Installation

Requirements‚Äã

  • Node 10.13.0 or newer
  • NPM 6.4.1 or newer
  • Android Studio (if you want to develop for Android)
  • Xcode (if you want to develop for iOS/tvOS)
  • Tizen Studio (if you want to develop for Tizen)
  • WebOS SDK (if you want to develop for WebOS)
  • KaiOS SDK (if you want to develop for KaiOS)

Install ReNative CLI (rnv)‚Äã

$ npm install rnv -g

Create new app‚Äã

$ rnv new

Follow steps in the terminal

Run first app‚Äã

Run your first ios app

$ rnv run -p ios

Run your first android app

$ rnv run -p android

Run your first web app

$ rnv run -p web

... and so on

All app code is located in ./src directory
